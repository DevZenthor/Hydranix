# Hydranix Website

Site vitrine officiel du joueur Fortnite **Hydranix** développé avec **React + Vite + Bootstrap**.

## Aperçu

Ce projet présente :

- Profil du joueur
- Team : Reality Zoned
- Manager : Zenthor
- Liens réseaux sociaux
- Fortnite Tracker
- Design gaming noir / rouge
- Background animé avec pétales sakura

---

## Technologies utilisées

- React
- Vite
- Bootstrap
- CSS3
- React Icons

---

## Installation

```bash
npm install
npm run dev
